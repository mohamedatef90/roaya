## Script Overview

### Purpose
Introduce Roaya's new website using a Q&A format that presents common problems IT decision-makers face and demonstrates how Roaya solves them. The script emphasizes transparency, ROI, and proven results.

### Key Messaging
- **Primary Differentiator:** Egypt's first transparent IT pricing
- **Unique Tool:** Interactive ROI calculator
- **Competitive Advantage:** WorldPosta partnership (global + local)
- **Proven Results:** 40% cost reduction, 99.9% uptime
- **Local Expertise:** 24/7 Arabic and English support

### Target Audience
- IT Directors, CTOs, CEOs
- Mid-size to large enterprises (50-500 employees)
- Industries: Banking, Manufacturing, Healthcare, Retail

## Platform-Specific Adaptations

### Facebook Optimization

**Hook:** Start with the most relatable problem (pricing transparency)  
**Length:** 45-50 seconds (slightly shorter)  
**Visual Style:** 
- More dynamic cuts (every 2-3 seconds)
- Bold, high-contrast text overlays
- Mobile-optimized text size (larger fonts)
- Square format (1:1) for feed, vertical (9:16) for Stories

**CTA Emphasis:** 
- Stronger visual emphasis on website URL
- Include clickable link in post description
- Use Facebook's call-to-action button overlay

**Captions:** 
- Auto-captions enabled (essential for Facebook)
- Text should be readable without sound

### YouTube Optimization

**Hook:** Slightly longer setup (5-7 seconds) to establish context  
**Length:** 55-60 seconds (full version)  
**Visual Style:**
- More cinematic transitions
- Can include more detailed visuals
- 16:9 aspect ratio (standard YouTube)

**CTA Emphasis:**
- Include clickable annotation/end screen
- Website URL in video description (first line)
- End screen with subscribe + website link

**SEO Elements:**
- Title: "Roaya IT: Egypt's First Transparent IT Services Pricing | Q&A"
- Description should include keywords: "IT services Egypt", "transparent pricing", "cloud hosting Egypt"

## Production Notes

### Voiceover Style
- **Tone:** Professional, confident, approachable
- **Pace:** Moderate (not rushed, but energetic)
- **Accent:** Clear English (Egyptian accent acceptable if natural)
- **Emphasis:** 
  - "FIRST" (emphasize uniqueness)
  - "40%" (emphasize results)
  - "roaya.co" (clear pronunciation)

### Music
- **Style:** Modern, upbeat, slightly corporate
- **Tempo:** Moderate (120-130 BPM)
- **Mood:** Confident, forward-moving
- **Volume:** Subtle background (voiceover is primary)
- **Transitions:** Music should build slightly at CTA
- **Recommendations:** 
  - Epidemic Sound: "Corporate Success" category
  - Artlist: "Business" or "Tech" playlists

### Visual Style Guidelines
- **Typography:** 
  - Bold, sans-serif (Montserrat, Inter, or similar)
  - High contrast (white on dark, or dark on light)
  - Mobile-readable (minimum 24pt for body text)
- **Animations:**
  - Smooth transitions (0.3-0.5s)
  - Numbers count up (not instant)
  - Logo appears with subtle scale animation
- **Color Palette:**
  - Primary: Roaya brand blue (#0066CC or brand standard)
  - Accent: Success green for positive metrics
  - Background: Clean white or dark (brand-appropriate)
  - Text: High contrast for readability

### Technical Specifications
- **Resolution:** 1080p minimum (4K preferred for YouTube)
- **Frame Rate:** 30fps
- **Aspect Ratios:**
  - 16:9 (YouTube, Facebook feed)
  - 1:1 (Facebook square, Instagram)
  - 9:16 (Facebook/Instagram Stories, Reels)
- **Audio:**
  - Voiceover: Clear, professional recording
  - Music: Royalty-free, properly licensed
  - Mix: Voiceover at -3dB, music at -12dB

## Notes for Director/Producer

### Critical Elements
1. **Opening Hook:** Must grab attention in first 3 seconds (Facebook algorithm)
2. **Visual Clarity:** All text must be readable on mobile devices
3. **Pace:** Keep energy high but not rushed (viewers need to process information)
4. **CTA Clarity:** Website URL must be visible and memorable
5. **Brand Consistency:** All visuals must match Roaya brand guidelines

### Common Pitfalls to Avoid
- Don't make it too salesy (focus on education, not hard sell)
- Don't use jargon (keep language accessible)
- Don't rush through key points (give numbers time to register)
- Don't forget mobile viewers (test on phone screen)
- Don't skip captions (essential for Facebook)

### Testing Recommendations
- Test with 5-10 target audience members before final production
- A/B test different hooks (pricing vs. ROI vs. support)
- Test both English and Arabic versions if budget allows
- Monitor first 24 hours of performance and adjust if needed
