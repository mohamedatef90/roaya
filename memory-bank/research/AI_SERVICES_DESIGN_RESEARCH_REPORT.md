## Executive Summary

This comprehensive analysis of leading AI company websites (OpenAI, Anthropic, NVIDIA, Runway ML, DeepMind) reveals a convergence toward sophisticated, performance-oriented design systems that emphasize:

1. **Fluid, responsive typography** using variable fonts and clamp-based scaling
2. **Minimalist color palettes** dominated by dark themes with strategic accent colors (warm terracotta, NVIDIA green #76b900, neural blues)
3. **Motion as intelligence**: Scroll-triggered animations, morphing transitions, and particle effects convey AI's adaptive nature
4. **3D/WebGL integration** for hero sections using Three.js, creating immersive entry points
5. **Glassmorphism over neumorphism** for modern, layered interfaces with backdrop blur effects
6. **Performance-first approach**: CSS custom properties, Intersection Observers, and reduced motion support

```javascript
gsap.to(".element", {
  scrollTrigger: {
    trigger: ".section",
    scrub: true,        // Smooth scroll sync
    start: "top bottom",
    end: "bottom top"
  },
  y: "-30%",           // Parallax movement

```

**Performance Patterns:**
- `will-change: transform` for animated elements
- `prefers-reduced-motion` support (critical)
- Animate `transform` and `opacity` only (not layout properties)
- Use `requestAnimationFrame` for custom animations

### Quick Start: Three.js Hero Section

**Installation:**
```bash
npm install three
```

**Basic Setup (300 lines for custom animation):**
```javascript
import * as THREE from 'three';

// Scene setup
const scene = new THREE.Scene();
const camera = new THREE.PerspectiveCamera(75, window.innerWidth / window.innerHeight, 0.1, 1000);
const renderer = new THREE.WebGLRenderer({ alpha: true, antialias: true });

renderer.setSize(window.innerWidth, window.innerHeight);
document.getElementById('hero').appendChild(renderer.domElement);

// Particle system (Neural Network Effect)
const particlesGeometry = new THREE.BufferGeometry();
const particlesCount = 5000;
const posArray = new Float32Array(particlesCount * 3);

for(let i = 0; i < particlesCount * 3; i++) {
  posArray[i] = (Math.random() - 0.5) * 5;
}

particlesGeometry.setAttribute('position', new THREE.BufferAttribute(posArray, 3));

const particlesMaterial = new THREE.PointsMaterial({
  size: 0.005,
  color: '#3c339a',
  transparent: true,
  opacity: 0.8
});

const particlesMesh = new THREE.Points(particlesGeometry, particlesMaterial);
scene.add(particlesMesh);

camera.position.z = 2;

// Animation loop
function animate() {
  requestAnimationFrame(animate);
  particlesMesh.rotation.y += 0.001;
  renderer.render(scene, camera);
}

animate();

// Responsive resize
window.addEventListener('resize', () => {
  camera.aspect = window.innerWidth / window.innerHeight;
  camera.updateProjectionMatrix();
  renderer.setSize(window.innerWidth, window.innerHeight);
});
```

### Quick Start: GSAP ScrollTrigger Parallax

**Installation:**
```bash
npm install gsap
```

**Implementation:**
```javascript
import { gsap } from "gsap";
import { ScrollTrigger } from "gsap/ScrollTrigger";

gsap.registerPlugin(ScrollTrigger);

// Simple parallax effect
gsap.to(".bg-layer-1", {
  scrollTrigger: {
    trigger: ".parallax-section",
    start: "top top",
    end: "bottom top",
    scrub: true,        // Smooth scroll sync
  },
  y: "-30%",
  ease: "none"
});

// Multi-layer parallax
const layers = gsap.utils.toArray('.parallax-layer');
layers.forEach((layer, index) => {
  const depth = layer.dataset.depth || 1;
  gsap.to(layer, {
    scrollTrigger: {
      trigger: ".parallax-container",
      start: "top bottom",
      end: "bottom top",
      scrub: true,
    },
    y: (i, target) => -ScrollTrigger.maxScroll(window) * target.dataset.speed,
    ease: "none"
  });
});

// Fade-in on scroll
gsap.utils.toArray('.fade-in').forEach(elem => {
  gsap.from(elem, {
    scrollTrigger: {
      trigger: elem,
      start: "top 80%",
      toggleActions: "play none none reverse"
    },
    opacity: 0,
    y: 50,
    duration: 1
  });
});
```

import * as d3 from 'd3';

const width = 800;
const height = 600;

const svg = d3.select('#neural-viz')
  .append('svg')
  .attr('width', width)
  .attr('height', height);

// Define network structure
const network = {
  layers: [
    { nodes: 3, label: 'Input' },
    { nodes: 5, label: 'Hidden 1' },
    { nodes: 5, label: 'Hidden 2' },
    { nodes: 2, label: 'Output' }
  ]
};

const layerSpacing = width / (network.layers.length + 1);

// Draw connections
network.layers.forEach((layer, layerIndex) => {
  if (layerIndex < network.layers.length - 1) {
    const nextLayer = network.layers[layerIndex + 1];

    for (let i = 0; i < layer.nodes; i++) {
      for (let j = 0; j < nextLayer.nodes; j++) {
        svg.append('line')
          .attr('x1', (layerIndex + 1) * layerSpacing)
          .attr('y1', (i + 1) * (height / (layer.nodes + 1)))
          .attr('x2', (layerIndex + 2) * layerSpacing)
          .attr('y2', (j + 1) * (height / (nextLayer.nodes + 1)))
          .attr('stroke', '#3c339a')
          .attr('stroke-width', 1)
          .attr('opacity', 0.3);
      }
    }
  }
});

// Draw nodes
network.layers.forEach((layer, layerIndex) => {
  for (let i = 0; i < layer.nodes; i++) {
    svg.append('circle')
      .attr('cx', (layerIndex + 1) * layerSpacing)
      .attr('cy', (i + 1) * (height / (layer.nodes + 1)))
      .attr('r', 15)
      .attr('fill', '#5fb7cf')
      .attr('stroke', '#3c339a')
      .attr('stroke-width', 2);
  }
});

### Key Concepts Summary

**1. Three.js for 3D:**

- Particle systems for neural network effects

**2. GSAP ScrollTrigger:**

**3. Particles:**

- Performance: limit particles on mobile

**4. Glassmorphism:**
- `backdrop-filter: blur()` is key

**5. D3.js Visualization:**

- **Particle Count**: More than 100 particles can impact mobile performance
- **D3.js Learning Curve**: Steep initial learning; start with simple examples
- **Browser Support**: Check caniuse.com for backdrop-filter, WebGL support
- **Reduced Motion**: ALWAYS respect prefers-reduced-motion media query

```javascript
// Simplified neural network visualization
const createNeuralViz = () => {
  const width = 800;
  const height = 400;

  const svg = d3.select('#neural-network')
    .append('svg')
    .attr('viewBox', `0 0 ${width} ${height}`)
    .attr('preserveAspectRatio', 'xMidYMid meet');

  const network = [
    { nodes: 3, color: '#9bffe0' },  // Input
    { nodes: 6, color: '#7e8bff' },  // Hidden 1
    { nodes: 6, color: '#7e8bff' },  // Hidden 2
    { nodes: 2, color: '#6fd4ff' }   // Output
  ];

  const spacing = width / (network.length + 1);

  // Draw connections with animation
  network.forEach((layer, i) => {
    if (i < network.length - 1) {
      const nextLayer = network[i + 1];
      for (let n1 = 0; n1 < layer.nodes; n1++) {
        for (let n2 = 0; n2 < nextLayer.nodes; n2++) {
          svg.append('line')
            .attr('x1', (i + 1) * spacing)
            .attr('y1', (n1 + 1) * (height / (layer.nodes + 1)))
            .attr('x2', (i + 2) * spacing)
            .attr('y2', (n2 + 1) * (height / (nextLayer.nodes + 1)))
            .attr('stroke', '#3c339a')
            .attr('stroke-width', 1)
            .attr('opacity', 0)
            .transition()
            .delay(i * 200 + (n1 * 50))
            .duration(500)
            .attr('opacity', 0.2);
        }
      }
    }
  });

  // Draw nodes with pulse animation
  network.forEach((layer, i) => {
    for (let n = 0; n < layer.nodes; n++) {
      const node = svg.append('circle')
        .attr('cx', (i + 1) * spacing)
        .attr('cy', (n + 1) * (height / (layer.nodes + 1)))
        .attr('r', 0)
        .attr('fill', layer.color)
        .attr('stroke', '#14192b')
        .attr('stroke-width', 2);

      node.transition()
        .delay(i * 200 + (n * 50))
        .duration(400)
        .attr('r', 12);

      // Pulse animation
      node.append('animate')
        .attr('attributeName', 'r')
        .attr('values', '12;14;12')
        .attr('dur', '2s')
        .attr('repeatCount', 'indefinite');
    }
  });
};

// Trigger on scroll
ScrollTrigger.create({
  trigger: '#neural-network',
  start: 'top 70%',
  once: true,
  onEnter: createNeuralViz
});
```

const loadThreeJs = async () => {
  if ('IntersectionObserver' in window) {
    const observer = new IntersectionObserver((entries) => {
      entries.forEach(entry => {
